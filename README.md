A function of the form
*y* = *f*<sub>1</sub>(*x*<sub>1</sub>) + *f*<sub>2</sub>(*x*<sub>2</sub>)
or
*y* = *f*<sub>1</sub>(*x*<sub>1</sub>)*f*<sub>2</sub>(*x*<sub>2</sub>)
can be handled with a 2-layer slide rule, while adding another layer
permits
*y* = *f*1(*x*<sub>1</sub>) + *f*2(*x*<sub>2</sub>) + *f*3(*x*<sub>3</sub>),
etc.
